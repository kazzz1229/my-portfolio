# フリマサイト

## 概要
- ユーザー登録、編集、退会、パスワード変更  
- 商品の出品、編集  
- お気に入り  
- 商品一覧のソート、絞り込み  

などの機能を作成

## 開発環境
インフラ：AWS  
コンテナ：Docker  
使用言語：HTML、CSS、JavaScript、PHP  
データベース：MySQL  
Webサーバー：Apache  
メールサーバー：Postfix
